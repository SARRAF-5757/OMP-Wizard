## To get going...
1. Clone repository
2. ```mkdir build```
2. ```cd build```
3. ```cmake ..```
4. ```make```
5. ```./omp-wizaard```

## References
- TUI library used - [FTXUI](https://github.com/ArthurSonzogni/FTXUI) - [Docs](https://arthursonzogni.github.io/FTXUI/index.html)

- Find [Diamonds](https://www.nerdfonts.com/cheat-sheet)
